# XR ready project

Готовый self-hosted WebXR шаблон со следующими функциями:

- поиск плоскости перед размещением
- reticle / прицел
- tap to place
- кнопка «Переставить»
- кнопка «Фото»
- Web Share API для шаринга фото
- музыка
- кнопка перехода на сайт
- деплой как обычный статический сайт

## Что заменить сразу

В `app.js` замени:

```js
const WEBSITE_URL = 'https://example.com'
```

на свой URL.

## Где поменять музыку

Замени файл:

`assets/music.wav`

на свой mp3/wav с тем же именем
или укажи другой путь в `AUDIO_URL`.

## Где поменять 3D

Сейчас подключены:

- `assets/textured_mesh.glb`
- `assets/textured_mesh_1.glb`

Можно заменить их своими файлами и, если нужно, поправить scale/position в `loadModels()`.

## Как деплоить на Render

1. Создай новый Static Site
2. Подключи GitHub репозиторий
3. Build Command: оставь пустым
4. Publish Directory: `.`
5. После деплоя открой ссылку с мобильного телефона

## Важно

Этот шаблон использует чистый WebXR через three.js.
Для лучшей совместимости нужен современный мобильный браузер с поддержкой WebXR/hit-test.
