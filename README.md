# XR project

Что уже сделано:
- WebAR по ссылке без установки
- поиск поверхности через `XR8.XrController.hitTest(...)`
- прицел по центру экрана
- кнопка размещения / перестановки
- фото через `XR8.CanvasScreenshot.takeScreenshot()`
- share menu, если браузер поддерживает `navigator.share({files})`
- кнопка перехода на сайт
- музыка

Что заменить перед публикацией:
1. В `app.js` поменяй:
   - `const WEBSITE_URL = 'https://example.com'`
2. При желании поменяй:
   - `MODEL_PATHS`
   - `assets/music.wav`

Как публиковать:
1. Загрузи весь архив в GitHub
2. На Render выбери **Static Site**
3. Публикуй корень проекта
4. Открывай только по `https://...`

Если на iPhone или Android не открывается камера:
- проверь HTTPS
- проверь разрешение на камеру
- лучше открывать в Safari / Chrome напрямую, а не из встроенного браузера соцсети
